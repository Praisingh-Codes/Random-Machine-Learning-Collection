## Random Machine Learning Collection

A curated collection of Machine Learning & Computer Vision projects covering prediction, clustering, classification, image recognition, gesture detection, and food calorie estimation.

This repository contains 5 end-to-end Machine Learning/CV projects, each structured with datasets, models, logs, Jupyter notebooks, and interactive Gradio apps.

#### Projects Included

Below is a quick overview of each project with its purpose and capabilities.

#### 1️⃣ House Price Prediction

A regression-based ML system that predicts house prices using numerical & categorical features.
Includes preprocessing, model comparison, evaluation metrics, and a Gradio-based prediction interface.

Key Features:

- Data cleaning & encoding

- Train/test split & model training

- Multiple regression algorithms

- Interactive price prediction UI

#### 2️⃣ Customer Segmentation using K-Means

A clustering system that groups customers into meaningful segments using K-Means.

Key Features:

- Preprocessing & scaling

- Elbow method for optimal K

- Cluster visualization

- Gradio interface for real-time cluster prediction

#### 3️⃣ Pet Image Recognition (Cat vs Dog Classifier)

A deep-learning classifier that identifies whether an image contains a cat or a dog.

Key Features:

- Custom CNN architecture

- Training logs & performance metrics

- Saved model for deployment

- Gradio upload interface for instant predictions

#### 4️⃣ Hand Gesture Recognition System

A ML-based system that recognizes hand gestures from uploaded images or webcam snapshots.

Key Features:

- PCA for dimensionality reduction

- SVM, KNN, and Random Forest models

- Webcam capture + upload-based prediction

- Real-time gesture recognition window

#### 5️⃣ Food Image Classifier & Calorie Estimator

A food recognition system that identifies food items and estimates their calorie content using a custom mapping.

Key Features:

- CNN feature extractor + SVM classifier

- Custom calorie JSON file

- Top-3 predicted food classes

- Gradio interface for food recognition & calorie estimation
